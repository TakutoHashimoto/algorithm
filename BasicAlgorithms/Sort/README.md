# ソートのアルゴリズム
**太字**のものは重要
* Bogoソート
* **Bubbleソート**
    * Cocktailソート（カクテルソート、シェーカーソート）
    * Combソート
    * **Selectionソート（選択法）**
    * Gnomeソート
* **Insertionソート（挿入法）**
    * Bucketソート
    * Shellソート
* Countソート
    * Radixソート
* **Quickソート**
* **Mergeソート**
* **Heapソート**
* Timソート
    * insertionソートとmergeソートを使用している
